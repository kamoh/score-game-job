About

I'm a software engineer at BeenVerified, the leading destination for easy and affordable access to public record information.

I also write for Hiveswap, What Pumpkin Studios' upcoming adventure game based on the popular Homestuck webcomic, and am a founding member of indie game developer Studio Mercato.

I like clean user experiences, narrative-driven games, and using language effectively to convey new ideas.
